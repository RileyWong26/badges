# Badges

Get badge images for your READMEs, or to be used anywhere.


## Endpoints
### Reposize
Returns an svg badge that has the size of your repo
#### Endpoint
```
  GET /repo-size/{your_github_name}/{repo_name}   `
```
#### Headers
``` 
headers: {
  'Authorization' : 'Bearer your_github_token_here'
}
```
